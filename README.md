# SpendWise – Smart Expense Tracker

SpendWise is a Python-based command-line application that helps users record, organize, and analyze daily expenses. It stores expense records in a CSV file and provides useful summaries based on categories and months.

## Features

- Add new expenses
- View all recorded expenses
- Calculate total spending
- Category-wise expense summary
- Monthly expense summary
- CSV-based data storage
- Simple menu-driven interface

## Technologies Used

- Python
- CSV
- File Handling
- Datetime Module

## Project Structure

```
SpendWise/
│
├── main.py
├── expenses.csv
├── README.md
└── requirements.txt
```

## How to Run

1. Clone this repository.
2. Open the project folder.
3. Run:

```
python main.py
```
